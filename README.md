# TaskMint 🚀

TaskMint is a premium To-Do List and Expense Manager app built with Flutter. It features a modern 3D Glassmorphism UI, smooth animations, and local data persistence.

## Key Features
- **To-Do List**: Manage daily tasks with image attachments.
- **Expense Tracker**: Track spending with category-wise summaries.
- **Modern UI**: 3D-inspired design with premium Glassmorphism cards.
- **Dark & Light Mode**: Seamless theme switching.
- **Local Storage**: All data is saved on your device using SharedPreferences.

## Getting Started
1. Clone the repository.
2. Run `flutter pub get` and `flutter run`.

## Tech Stack
- Flutter & Dart
- Provider (State Management)
- SharedPreferences (Local Storage)
- Google Fonts (Outfit)
